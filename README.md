# EFI-Catalina-OC-GA-Z97X-UD3H
OpenCore EFI files for OSX Catalina on GA-Z97X-UD3H
This repository contains EFI files required for my OpenCore 0.8.0 boot setup with MacOS Catalina 10.15.7.
